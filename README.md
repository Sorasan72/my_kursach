# Курсовой проект "Калькулятор для путешественника"

Веб-приложение "Калькулятор для путешественника" предназначено для конвертации валют, измерений длины и подсчёта итоговой суммы бюджета. Будет полезным для гидов, организаторов поездок, туристов.
<hr>

## Функционал: ##
Интерфейс состоит из трёх основных частей:
- Ковертация валют:
    * выбирается валюта ИЗ которой нужно конвертировать, затем В какую нужно конвертировать и вводится сумма числом(!);
    * после нажатия на кнопку "Конвертировать" полученная сумма будет выведена на экране.
- Конвертация измерений длины:
    * конвертируются мили в километры и наоборот; при нажатии на соответствующие радио-кнопки и вводе нужного значения на экран выведется итоговая величина.
- Расчёт бюджета поездки:
    * форма, позволяющая создать список с названием и суммой расхода с возможностью добавлять и удалять пункты, а затем подсчитать итоговую сумму;
    * добавить пункт можно с помощью полей ввода над списком по умолчанию и кнопки "Добавить пункт", удалить пункт можно с помощью кнопки с иконкой корзины, очистить весь список - по нажатию на кнопку "Очистить";
    * нажав на кнопку "Рассчитать бюджет" на экран выводится таблица со значениями, ранее вписанными пользователем. Итоговая сумма выводится в рублях.
